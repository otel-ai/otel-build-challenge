# Contributing

Thanks for building on this challenge brief.

## Faster setup (optional)

Many teams copy the [`.otel-scaffold/`](.otel-scaffold/) directory into their
solution repository root for Cursor / agent conventions, progress tracking, and
starter scripts. It is not required and is not listed in the main README file
list — use it only if it helps your workflow.

## Solution repository

- Do **not** fork this brief repo.
- Keep `ATTESTATION.md`, `etl/LOAD_PROOF.json`, tools, tests, skills, and
  `ARCHITECTURE.md` in your own repo.
- Open a LinkedIn DM for Phase 0 ETL review — we do not accept drive-by PRs to
  this repository.
